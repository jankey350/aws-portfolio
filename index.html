AWS AgentCore Portfolio

Secure AI Ticket Routing System

A production-style AI application built using Amazon Bedrock AgentCore, Amazon Bedrock Knowledge Base, and CrewAI. The application is deployed inside a custom VPC with private subnets and uses VPC Endpoints for secure communication with AWS services.
